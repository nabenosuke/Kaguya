# ＜輝夜の城で忍びたい(仮称)＞
#ゲーム説明
【遊び方】
"KaguyaVerXX"という名前のzipファイルを解凍し、
"Kaguya(.exe)"という名前のアプリケーションを実行してください。


[←][↑][↓][→]キー：移動
[z]キー:ジャンプ、決定
[x]キー:攻撃、キャンセル



【確認不具合一覧】
・タイトルで何もない場所にマウスをクリックしたら選択が外れる。
→マウスで任意の項目をクリックすれば戻ります。



【実装予定の要素一覧】
・操作キャラの追加
・ボスの追加
・ポーズ画面
・スコア
・中間ポイント
・すり抜け床
・背景に小道具
・背景の色をキャラ毎に見やすい色に変更
・クリア特典
